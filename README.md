### Как установить
- Скачайте код.
- Установите виртуальное окружение командой: 
```
python -m venv venv
``` 
- Запустите виртуальное окружение командой:  
```
venv\Scripts\activate
``` 
- Установите зависимости командой:  
```
pip install -r requirements.txt
```
